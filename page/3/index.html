登山之習

发表于2023-09-26更新于2023-09-26

1 自由

还是从一些零碎的感受说起，匆匆翻过一遍斯宾诺莎论人自由的小书，现写下自己试图理解他在讲什么时保留的粗浅痕迹。

一个明显的直觉是斯宾诺莎和孔子的气质很像，以至于看书时恍惚眼前是论语，只把爱慕上帝替换为追求完满的君子，几乎可以贯通。不同，孔子不讲什么推理只说如何做。斯宾诺莎试图在认知层次讲清楚去做的必然性。

人在和外物融合为一，体会到部分来自于上帝的完善，积累这种完善，直到和上帝融合为一，走向真正的幸福。这里上帝是万物存在的第一因。

人爱恋完善的东西，当其越完善，爱恋也就越深刻。这种爱恋不是自由的，人对完善的爱和倾向也是被动的。

为什么人需要爱恋呢？因为人的本性是虚弱的，不和其他好的完善的东西联合来获取力量，是无法维系存在的。

进而，当人选择和暂有的仅仅含有很少完善的东西相结合时，一定会陷入不幸。如名利、权力、美色等就像大树的叶子，与叶子的爱恋的结合是脆弱的，它的枯萎会震动人的感情。叶子依赖大树的根存在，为什么不和这个日日翻新的树根联合呢？因为人看不见被遮蔽的东西。

人有几种方式来认知事物，道听途说的，例如日本对自己二战的罪行是死不悔改等，从日常的听闻中得来某种判断，这种判断多是缺少理性的盲目的意见，进而产生的憎恶等情感。整个被动的过程，怎么说？一切可以言说的东西和至高的完善相比都是无谓的东西，让人好奇有那么多的完善去实现，为什么浪费时间在这种被动的、盲目的情感上呢？即人可以通过悟性来避免这种错误的发生。

接着是从经验而来的意见，医生用过某种药物好用，就习惯认为这种药是万能的。悟性对从经验而来的意见表现无力，似乎从亲自感受出发的东西和悟性是平行的关系。

还有来自理性推理得到的真实的信仰，如追求完善是自然的，人应该去追求完善。注意到，这里的说法是应该而不是真实的，说明眼前的判断仍然在我之外，需要用文字来暂存这种认识，而这部分的应该是真实的一部分，或者是真实的一个影子。在人和这个真实合二为一之前，该认知只是一个推理，以及从推理而来的信念支撑人去追求完善的东西，当人进入完满的实相中，应该如何也就失去了作为阶梯或船的使命，剩下的是真相全体。

最后的方法是如实的见到实相并拥有明确的认知，即在那个东西的本身上见到那个东西，而非借助外部的描述的转手来认识这个东西。就是说人完全和上帝留存在这个东西中的完善合二为一，人享有了这个东西本身和清楚感知它的一切。

没有本质的东西是不可被思议的。躯体是一种运动和静止的比例。人的心灵是上帝的无限完善的思想属性的一个式态，依附于上帝这个根本的实体，心灵拥有认识不同式态的能力，并自然依恋于极致完善的实体。人自然的倾向是完善他人，并从他人处感知上帝的一部分完善。如此，直到能保有在极致的完善中，成为猎户手里合适的斧头。

人的安心和恐惧是一回事，都是不应该。因为对外物的好，产生了想要得到的希望，有把握就感到安心，否则恐惧，都是求之外，如果始终以追求完善为目标，日有寸进，何来忧惧和安心这些无所谓的感情呢？注意开头的不应该，该表达是一种典型的借助工具来说明，背后缺少对真实的完整把握，而仅仅借助理性推理得出某种情感是多余的。

意志也是思想的某种式态，谈不到自由，本身就是无，无如何生无呢？即只有实体才谈得上自由，他们都是实体产生的一种式态。同样的人的心灵也是上帝思想的一种式态，也是一种无，谈不上产生新的式态，如自由。

可以说，人完全被动的，但这种被动是更为深刻的自发倾向，即人只是自发的趋于完善，并把不完善的回归完善。

怎么做才能保证如水常通流的强劲动力？

如何才能做到时时戒惧不息，刚毅向前？

  • 圣人在前。子绝四：勿意，勿必，勿固，勿我。
  • 树立目标：一定要将所有的私欲遮蔽扫除干净。
  • 中道而立，能者从之。君子引而不发，跃如也。
  • 应无所住，而生其心。
  • 依般若波罗蜜多故，心无挂碍。

2 扩充不忍摧抑生机之情

  • 感受生机活泼泼才是正，才是所当然之理。

  • 人心有四端：恻隐之心，仁之端也；羞恶之心，义之端也；辞让之心，礼之端也；是非之心，智之端也。

  • 用灵魂去体会，化身为各色生灵，并克制用文字（概念）去描述的冲动。

  • 养活，养护一个具体的生命。

  • 生命真神：从理性中推导可知，眼耳鼻舌身意背后有一个真神。
    • 自性，良知，真宰
    • 仁的本体
    • 心的灵明
  • 向具体的对象忏悔，形成一种完整的体验。
    • 隐性的，慎独要面对的就是自己的良知。

    • 故要抓住一切的时间去全观那个至高的没有遮蔽的真神。

仁是什么？

  1. 躬行而知未尽，曰仁曰诚。

    1. 轩昂奋发，收拾身心，自作主宰。心灵完全是主动的，向上的。

  2. 刚毅木讷近乎仁，仁者必在刚。

    1. 阳，刚直之物，全用刚直，非天下至理莫能及。
    2. 识得所当然之理，必刚直而更进一步，力达知未尽。

  3. 仁是一团生生不息的和气，总是安和泰然的样子。

    1. 本来的生活就在当下，自然就能不断翻新，意味悠长。
    2. 不迁怒（过而不留），纯然以【应无所住而生其心】的态度观照万事万物。
    3. 没有一丝一毫私欲的病痛，纯然是通达的。

  4. 仁是与万物浑然一体，义礼智信皆仁也；仁是当下明锐的直觉。

    1. 吾心如天渊、如明镜、如流水，能进入万物同然之枢纽。
    2. 以明锐直觉，观其节限、分寸、先后，方尽物之性，安于自然。

  5. 仁者重心完全在当下的生活，实现自然的生命之理，完全无所依待。

    1. 总是安和自在，种种不成问题，时时是乐。
    2. 内在生机自然就是不断翻新的，自然就是调和畅达的。

  2. 里面乱，心不能安，牵挂得失。
  3. 懈怠，白天睡觉，没有上进的活力。
  4. 向外找快乐，不能安于当下，总向外找刺激。
  5. 向外找道理，但无法合于心，难以强行，或其进锐者其退速。

真宰是什么？

有意识与无意识

  • 有意识的做事
    • 先A就有B，基于知识理论，进行逻辑分析。
    • 存有一个等待客观结果到来的意思，算是理性的预判。
    • 有所为而为，做之前就存了一个待定的、期盼的结果。
    • 可以用概念，符号来准确描述其过程。
  • 无意识的做事
    • 做A，只是一种它很合适的意味。
    • 是主观的情理，像真的心所发的直觉。
    • 无所为而为，不为什么就是自然的发动。
    • 往往无法用言语表达其内在原理。
  • 工具与主宰者
    • 理性思考的头脑是工具，他能得出趋利避害的应该如此。
    • 保存自我、繁衍后代的本能也是工具，他提供了一个实在的躯壳。
    • 先人的知识、概念、理论更是工具，它给出了可能有用的各种经验。
    • 存在真的心，名之曰真宰，能够不为什么的调用这些工具。
  • 不为什么的背后
    • 是自然，是必然，直觉得就该如此。
    • 与人相与，自然就感觉公平是合适的态度。
    • 有一种万物之所同然的所当然之理，各得其所，都合适。
  • 元神和识神
    • 面对一件事、一个物体，对其进行概念的定名就是识神，是一个分别心，即定名，分析和比较。
    • 元神是超越概念和符号的，讲不出来他是什么，也不需要讲出来，就是觉得对，顶合适。

真宰不是什么？

  1. 享乐的，有明确物质得失的要怎样。
  2. 作为动物，需要吃喝的本能所限。
  3. 没当然道理的、过度的、放纵的欲望。
  4. 不舒展的滞涩中，由着眼耳鼻舌身意等个体调用元气。

从保存自我上发出的声音

生活中常常会遇到拖延的情形，为什么会拖延呢？

仔细去看这些被拖延的念头，有个普遍的语言格式：再不去做A那么B就得不到了。

简单说，当下要做的事情总是未来某个目标的手段。这个目标常常和具体的自我保存相关，如收入、名利等。

拖延的态度很有意思：当为自己的保存争取更多可用的财货，得到的安全感也越强，某种程度上，也是自我保存（本能）的自然倾向，即它是应该的；但面对这样一个应该去做的声音，往往内心存在抵触，所以究竟在抵触什么呢？

子曰：“富而可求也，虽执鞭之士，吾亦为之。如不可求， 从吾所好。”能从正道上赚钱再贫贱、辛苦也可以做。如果求不到，不如致力于道。这样看，追求合理的财富本身是没有问题的，那它的排斥，道理在哪里？是不是我的过度反应呢？

回过头问，我排斥的对象究竟是什么？

功利的态度，即未来总有个一定要得到的东西让自己悬着心，害怕不能得到，所谓有所忧虑则不得其正。所以，我要问追求必要的生活资料一定得患得患失吗？这个一定要经得住仔细考察吗？

说白了，在追求财货中，因为意见从【自我保存】上起来，总带着一种过度的、非得不可的【偏执】，区别于从真宰发出的【无意识】直觉的简单直接，此意见总是纷纷扰扰的计较不清。

问题出在自己没有给【赚钱】这件事一个正确的态度。或者说一直以来这个态度都是从【自我保存】上来，难免带着一种【过度的偏执】，当自己厌恶此种【心不得正】的偏执时，顺带对【赚钱】也产生了不合理的排斥。

于是可以看出从【自我保存】出发的种种【应该如此】是一个不怎么干净的态度，在本来简明直接的【事物本身】上附带了许多【莫须有的期必】，使得这个应该变得难以执行，往往一拖再拖。

问题更进一步，为什么【自我保存】可以绕过真宰直接发出一个【要不这样就不行】的指令呢？

  • 推给旧习气。因为以前多是依靠【自我保存】的本能去判断价值，去做事。
    • 向外找成功的种种方法，偏于枝叶，而没有扎根。
    • 侧重于头脑简单的趋利避害，而非心灵的舒展自然。
  • 早期真宰虚弱。自接触六祖坛经后才知道原来身体内还有【更高级的主宰者】，有不为什么而做事的可能，经过一段时间有意识的收敛元气，才逐渐看清楚了心是地，性是王，性在王在，性去王无。
  • 良知被遮蔽的厉害。甚至于不能听到从良知发出的好恶判断，只能任由元气乱流，四处耗散，损伤心神。
    • 从趋利避害的角度看，耳目也是失去真宰的受害者，因为没有真宰坐镇来合理的疏导元气，使得元气都从眼睛（出口大）冲出体外，使其酸胀不能视物。
    • 没有良知分辨，只能依靠头脑对知识的逻辑推理，做出一个符合【自我保存】的方案，然而这个【期必】的方案无形中被良知所抵触。即内在的心灵是自我矛盾而混乱的。

简单说，【自我保存】以前作为太后干预政治，现在【真宰】成长起来了，需要将权力从太后的手里收回来，完全的交给【真宰】来处理正事。怎么收回来呢？

  • 定好名分，【自我保存】退居为工具，负责保存元气、收敛精神。
  • 事事交到【真宰】面前来考察，从无所为而为的立场出发。

  • 对事物按照其本来的样子看待，不掺杂人主观的意志。

疑问

1 若说合，如何景象？

  • 良知良能是本心，昏昧放逸是习心。向来不识此理，故种种本心为习心用；今来既识此理，故种种习心为本心转。

    • 习心，趋利避害（好逸恶劳）的自我保存本能。

  • 当下认取，活泼泼地，不须着纤毫气力。

  • 盖其间调停节候，如鸟之肄飞，冲然自得，便是『说』也。

2 怎么才能合于吾心呢？

  • 勿忘，即不要松懈懒怠；勿助，即不要急躁冒进。

  • 人性皆善，而觉有先后，后觉者必效先觉之所为，乃可以明善而复其初也。

真宰是什么？

有意识与无意识

  • 有意识的做事
    • 先A就有B，基于知识理论，进行逻辑分析。
    • 存有一个等待客观结果到来的意思，算是理性的预判。
    • 有所为而为，做之前就存了一个待定的、期盼的结果。
    • 可以用概念，符号来准确描述其过程。
  • 无意识的做事
    • 做A，只是一种它很合适的意味。
    • 是主观的情理，像真的心所发的直觉。
    • 无所为而为，不为什么就是自然的发动。
    • 往往无法用言语表达其内在原理。
  • 工具与主宰者
    • 理性思考的头脑是工具，他能得出趋利避害的应该如此。
    • 保存自我、繁衍后代的本能也是工具，他提供了一个实在的躯壳。
    • 先人的知识、概念、理论更是工具，它给出了可能有用的各种经验。
    • 存在真的心，名之曰真宰，能够不为什么的调用这些工具。
  • 不为什么的背后
    • 是自然，是必然，直觉得就该如此。
    • 与人相与，自然就感觉公平是合适的态度。
    • 有一种万物之所同然的所当然之理，各得其所，都合适。
  • 元神和识神
    • 面对一件事、一个物体，对其进行概念的定名就是识神，是一个分别心，即定名，分析和比较。
    • 元神是超越概念和符号的，讲不出来他是什么，也不需要讲出来，就是觉得对，顶合适。

真宰不是什么？

  1. 享乐的，有明确物质得失的要怎样。
  2. 作为动物，需要吃喝的本能所限。
  3. 没当然道理的、过度的、放纵的欲望。
  4. 不舒展的滞涩中，由着眼耳鼻舌身意等个体调用元气。

生活中常常会遇到拖延的情形，为什么会拖延呢？

仔细去看这些被拖延的念头，有个普遍的语言格式：再不去做A那么B就得不到了。

简单说，当下要做的事情总是未来某个目标的手段。这个目标常常和具体的自我保存相关，如收入、名利等。

拖延的态度很有意思：当为自己的保存争取更多可用的财货，得到的安全感也越强，某种程度上，也是自我保存（本能）的自然倾向，即它是应该的；但面对这样一个应该去做的声音，往往内心存在抵触，所以究竟在抵触什么呢？

子曰：“富而可求也，虽执鞭之士，吾亦为之。如不可求， 从吾所好。”能从正道上赚钱再贫贱、辛苦也可以做。如果求不到，不如致力于道。这样看，追求合理的财富本身是没有问题的，那它的排斥，道理在哪里？是不是我的过度反应呢？

回过头问，我排斥的对象究竟是什么？

功利的态度，即未来总有个一定要得到的东西让自己悬着心，害怕不能得到，所谓有所忧虑则不得其正。所以，我要问追求必要的生活资料一定得患得患失吗？这个一定要经得住仔细考察吗？

说白了，在追求财货中，因为意见从【自我保存】上起来，总带着一种过度的、非得不可的【偏执】，区别于从真宰发出的【无意识】直觉的简单直接，此意见总是纷纷扰扰的计较不清。

问题出在自己没有给【赚钱】这件事一个正确的态度。或者说一直以来这个态度都是从【自我保存】上来，难免带着一种【过度的偏执】，当自己厌恶此种【心不得正】的偏执时，顺带对【赚钱】也产生了不合理的排斥。

于是可以看出从【自我保存】出发的种种【应该如此】是一个不怎么干净的态度，在本来简明直接的【事物本身】上附带了许多【莫须有的期必】，使得这个应该变得难以执行，往往一拖再拖。

问题更进一步，为什么【自我保存】可以绕过真宰直接发出一个【要不这样就不行】的指令呢？

  • 推给旧习气。因为以前多是依靠【自我保存】的本能去判断价值，去做事。
    • 向外找成功的种种方法，偏于枝叶，而没有扎根。
    • 侧重于头脑简单的趋利避害，而非心灵的舒展自然。
  • 早期真宰虚弱。自接触六祖坛经后才知道原来身体内还有【更高级的主宰者】，有不为什么而做事的可能，经过一段时间有意识的收敛元气，才逐渐看清楚了心是地，性是王，性在王在，性去王无。
  • 良知被遮蔽的厉害。甚至于不能听到从良知发出的好恶判断，只能任由元气乱流，四处耗散，损伤心神。
    • 从趋利避害的角度看，耳目也是失去真宰的受害者，因为没有真宰坐镇来合理的疏导元气，使得元气都从眼睛（出口大）冲出体外，使其酸胀不能视物。
    • 没有良知分辨，只能依靠头脑对知识的逻辑推理，做出一个符合【自我保存】的方案，然而这个【期必】的方案无形中被良知所抵触。即内在的心灵是自我矛盾而混乱的。

简单说，【自我保存】以前作为太后干预政治，现在【真宰】成长起来了，需要将权力从太后的手里收回来，完全的交给【真宰】来处理正事。

怎么收回来呢？

  • 定好名分，【自我保存】退居为工具，负责保存元气、收敛精神。
  • 事事交到【真宰】面前来考察，从无所为而为的立场出发。

邵康节的观物态度

邵康节，看这人须极会处置事，被他神闲气定，不动声气，须处置得精明。 他气质本来清明，又养得来纯厚，又不曾枉用了心。他用那心时，都在紧要上用。 被他静极了，看得天下之事理精明。尝於百原深山中辟书斋，独处其中。王胜之 常乘月访之，必见其灯下正襟危坐，虽夜深亦如之。若不是养得至静之极，如何 见得道理如此精明！只是他做得出来，须差异。……想见 他看见天下之事，才上手来，便成四截了。其先後缓急，莫不有定；动中机会， 事到面前，便处置得下矣。康节甚喜张子房，以为子房善藏其用。以老子为得易 之体，以孟子为得易之用，合二者而用之，想见善处事。”问：“不知真个用时 如何？”曰：“先时说了，须差异。须有些机权术数也。

以物观物，性也（客观）；以我观物，情也（主观）。性公而明，情偏而暗。

  • 对事物按照其本来的样子看待，不掺杂人主观的意志。
  • 易地而处，换位思考，超越个人的私欲，从不同的角度上理解问题。

仁是什么？

  1. 躬行而知未尽，曰仁曰诚。

    1. 轩昂奋发，收拾身心，自作主宰。心灵完全是主动的，向上的。

  2. 刚毅木讷近乎仁，仁者必在刚。

    1. 阳，刚直之物，全用刚直，非天下至理莫能及。
    2. 识得所当然之理，必刚直而更进一步，力达知未尽。

  3. 仁是一团生生不息的和气，总是安和泰然的样子。

    1. 本来的生活就在当下，自然就能不断翻新，意味悠长。
    2. 不迁怒（过而不留），纯然以【应无所住而生其心】的态度观照万事万物。
    3. 没有一丝一毫私欲的病痛，纯然是通达的。

  4. 仁是与万物浑然一体，义礼智信皆仁也；仁是当下明锐的直觉。

    1. 吾心如天渊、如明镜、如流水，能进入万物同然之枢纽。
    2. 以明锐直觉，观其节限、分寸、先后，方尽物之性，安于自然。

  5. 仁者重心完全在当下的生活，实现自然的生命之理，完全无所依待。

    1. 总是安和自在，种种不成问题，时时是乐。
    2. 内在生机自然就是不断翻新的，自然就是调和畅达的。

不仁是什么？

  1. 麻木的、冷硬的、气血不通的样子。
  2. 里面乱，心不能安，牵挂得失。
  3. 懈怠，白天睡觉，没有上进的活力。
  4. 向外找快乐，不能安于当下，总向外找刺激。
  5. 向外找道理，但无法合于心，难以强行，或其进锐者其退速。

具体的仁

  1. 看着麻雀，自己仿佛也是他们中的一位，扭动灵活的脖颈，站在细细的树枝上，彼此拥挤着，并不做声。
