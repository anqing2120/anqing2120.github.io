虚其心，实其腹，弱其志，强其骨

经文

今时之人不然也，以酒为浆，以妄为常，醉以入房，以欲竭其精，以耗散其真，不知持满，不时御神，务快其心，逆于生乐，起居无节，故半百而衰也。

  1. 喝酒

    1. 不建议大家喝啤酒，你用啤酒炖鸡、炖鸭，倒是赞成
    2. 关于那些寒性的酒，我们不要碰，啤酒、葡萄酒少喝点儿可以，助助兴。烈性的白酒要少碰，有应酬可以稍微喝点儿。
    3. 喝酒的最高境界是什么呢？饮酒不醉为最高，喝酒喝到鼓舞一下兴致、微醺的时候打住，这叫“食饮有节”。
    4. 像喝水一样喝酒，于是平时不敢干的事敢干 ——“以妄为常”。接下来是疯和狂——“登高而歌， 弃衣而走”，老子天下第一。

  2. 现在的人需要的不多，想要的多。

    1. 总觉得别人成功，别人好，争着做别人。
    2. 没几个人有自知之明，想要做自己的。

  3. 悠着点儿用精

    1. 说精髓，肾主骨，骨生髓， 肾藏精，肾把精藏藏在骨子里。
    2. 还有一部分精气存在皮下脂肪里，抽脂是抑郁症的前奏。
    3. 性成熟后，老天会给你一种推动力，让你很有快感地去为了繁衍后代而死掉。（动物界有螳螂，大马哈鱼……）
    4. 精狭义是指生殖器官的精液，看似只有一点，但恢复圆满要花费巨大的能量。
    5. 各种漏精（气）
      1. 出盗汗。
      2. 喝酒过度用肾来解毒。
      3. 健身房过度出汗。

  4. 自己够强才去献血，一滴精十滴血，精血是同源的。

  5. 和老天对着干就是找倒霉

    1. 很多女性认为只要还来月经，说明自己还年轻。吃雌激素片，人为来月经当然是反自然。从中医来说，是用一种刺激的方法，让人透支自己的精髓再化成血，然后再流出来，即透支以后的生活来维持现状。结果呢？诱发乳腺癌。
    2. 天道是不以人的意志为转移的，你改变不了它。 人所能做的是根据天道左右自己的思想行为，如果你的思想行为跟天道、地理变化相适应，这就叫“有德”。
    3. 现在所谓的“喝牛奶补钙”更多是一种噱头。我们看一下它是不是自然的食品。动物长大了还回去跟它妈那喝奶吗？只有人，而且只有无知的人才会这么干。

  6. 该吃多少肉，该吃多少粮食

    1. 看恐龙化石，怎么判断是草食还是肉食？看牙齿。肉食恐龙都是尖锐的犬齿，草食恐龙全是臼齿（槽牙）。数数自己有几颗尖牙利齿，磨碎种子食物的槽牙有多少，比例大概是多少。
    2. 没有金刚钻别揽瓷器活。

  7. 科学

    1. 现代人说吃早饭有益健康，不吃早饭容易得胆结石。没有一个人 问，您早上起来饥吗？饿吗？想吃东西吗？昨天晚上吃的东西还没消化，打个嗝还是昨天吃饭的味，老婆就把牛奶鸡蛋端上来，还说这个东西科学，不吃不行。

    2. 第一句话：五谷为养。养的是您的精。第二句话：五畜为益。什么叫“益”？锦上添花。您在吃五谷的基础上加点儿肉食，别颠倒了。这是对我们中国人说的。第三句话：五菜为充。吃多了肉，吃些蔬菜疏导一下。没有五谷、没有肉吃的时候，权且充饥。最后 一句话：五果为助。

    3. 我的患者中，冬泳诱发心脏病的，冬泳导致自己长一身黑痦子的， 冬泳导致自己得抑郁症的例子，比比皆是。所以，如果你有长期养成的习惯，而且通过洗凉水澡能把自己的阳气激发出来，把寒气驱走，而不是把寒气留在体内，那么您适合冬泳；否则不要尝试，做个平常人就好。

持满：让肾精总是保持一种充盈状态而不是枯竭。

  1. 退化：学会了利用工具就丧失了本能。

    1. 有手机了，人基本记不住几个电话号码。
    2. 地震来之前，动物有反应，人没反应。
    3. 甲骨文、青铜器，到各种玉器的雕刻，还有秦汉以前精美的文章，你就会发现，其实人的本能在退化。
    4. 只能看到满目的人造灯光而看不到星星的时候，人离自然远了。
    5. 回归天真、纯朴、原始的状态，让我们对天地的感知更敏感。

  2. 养神

    1. 跟着自然的节奏去做事。
    2. 中医意识到情绪对身体节奏的影响，它说怒则气上，你一生气，气血就往上涌，怒发冲冠，脸红脖子粗；恐则气下，人受到惊吓，屁滚尿流的；惊则气乱，喜则气缓，思则气结，这些情绪其实都是扰乱了你的内心。
    3. 中医认为，人生不是痛苦的，从出生到生长、发育、死亡，其实是一种自然的过程，除非你违反了自然规律，有了病痛，才会觉得痛苦。
